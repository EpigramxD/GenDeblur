## Requirements:
- Python 3.8.0
- All packages from requirements.txt

## Installation:
```
pip install -r requirements.txt
pip install gendeblur
```

Build package: python -m build

Push package to PyPi: python -m twine upload --verbose --repository pypi dist/*
